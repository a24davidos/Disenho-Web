DESEÑO RESPONSIVO

Páxina desenvolvida polo alumnado do Ciclo de DAW para o módulo DIW, inspirada nos recursos de @openwebinars.

Establece dous puntos de ruptura, a 576px e a 992px, nos que se produce un cambio no deseño da páxina.

O proceso de desenvolvemento debe realizarse comezando polo deseño para pantallas pequenas.

PARA REPASAR

Concepto 1

Breakpoints

Debemos lembrar que o concepto de deseño responsivo, ademais de adaptación ao tamaño implica o cambio no deseño atendendo ao tamaño ou outras características do dispostivo. A anchura da pantalla na que se produce o cambio é o que se coñece como Breakpoint.

Concepto 2

Media Queries

Unha das ferramentas esenciais para poder realizar un deseño responsivo de calidade son as media- queries que son un módulo de CSS3 que nos permite adaptar a representación do contido ás características do dispositivo.

Concepto 3

Mobile First

Mobile First é un concepto de deseño web optimizado para móbiles, onde se crea inicialmente unha páxina web para pantallas pequenas (smartphones e tablets). Calquera proceso de optimización de páxinas web baseado na estratexia Mobile First está guiado polos principios do deseño web responsive e normalmente usa CSS Grids para que a páxina poida ser escalable.
